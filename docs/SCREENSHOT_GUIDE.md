## üìù Tips for Great Screenshots

1. **Clean Data**: Use meaningful example content (concepts about programming, science, etc.)
2. **Consistent Theme**: Use the same theme (dark/light) across all screenshots
3. **No Personal Info**: Don't show real email addresses or sensitive data
4. **Resolution**: Keep screenshots at least 1280px wide for clarity
5. **Annotations**: Consider adding arrows or highlights to key features (optional)
6. **Browser**: Hide browser UI (use fullscreen mode F11) or crop it out
